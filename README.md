# محول النص إلى صوت درامي (Dramatic TTS)

تطبيق لإنشاء حوارات صوتية درامية مبني باستخدام إطار عمل Flutter. يتيح التطبيق للمستخدمين كتابة سيناريوهات وحوارات، تعيين أصوات وشخصيات مختلفة، وضبط الأداء الدرامي مع دعم اللهجة الفصحى والعامية.

## الميزات الأساسية
- **تعدد الأصوات**: يتضمن مجموعة من الشخصيات المدمجة (رجل، امرأة، مراهق، بنت، طفل).
- **تمييز الألوان (Syntax Highlighting)**: تلوين النص تلقائيًا داخل المحرر بناءً على الشخصية التي تتحدث (مثال: `[رجل]` يظهر باللون الأحمر، `[امرأة]` بالأخضر).
- **دعم اللهجات**:
  - الفصحى: نطق سليم للنص المكتوب.
  - العامية القاهرية: معالجة النص تلقائياً (مثل تحويل حرف القاف إلى ألف باستثناء كلمات محددة مثل "القاهرة").
- **استنساخ الأصوات**: إمكانية رفع ملفات صوتية لاستنساخ الأصوات كميزة (صوت1 وصوت2).
- **ضبط الأداء الدرامي**: التبديل بين وضع هادئ ووضع حاد/انفعالي.
- **توليد الحوار**: محاكاة دمج الحوار مع فواصل زمنية وإمكانية تحميل الملف النهائي.

## التقنيات المستخدمة
- **Flutter** & **Dart**: لبناء واجهة المستخدم عبر منصات متعددة (Desktop, Web, Mobile).
- **file_picker**: لاختيار ملفات الصوت الخاصة بعملية استنساخ الأصوات المخصصة.

## كيفية التشغيل (التطوير المحلي)
1. تأكد من تثبيت [Flutter SDK](https://flutter.dev/docs/get-started/install).
2. قم باستنساخ المستودع وقم بتشغيل الأمر التالي لجلب الاعتمادات:
   ```bash
   flutter pub get
   ```
3. لتشغيل التطبيق على منصة سطح المكتب (مثلاً Windows):
   ```bash
   flutter run -d windows
   ```
4. لبناء ملف تنفيذي (.exe) للويندوز:
   ```bash
   flutter build windows
   ```
   ستجد الملفات الناتجة في `build/windows/runner/Release/`.

## حول CouldAI
تم إنشاء هذا التطبيق باستخدام [CouldAI](https://could.ai)، وهو أداة ذكاء اصطناعي لبناء تطبيقات متقاطعة المنصات. يحول CouldAI الأوصاف النصية إلى تطبيقات أصلية جاهزة للإنتاج (iOS، Android، Web، Desktop) مع وكلاء ذكاء اصطناعي مستقلين يقومون بهندسة، بناء، اختبار، ونشر التطبيقات بشكل متكامل.
